# Phase5B - Frontend Styles (CSS)

## 目标
根据设计规范建立统一样式：色彩、字体、阴影、间距、按钮与卡片。

## 强约束（必须执行）
- 任何代码文件不得超过 200 行。
- 每完成一个 checkbox 就立刻打勾。
- 必须严格遵守本文件指令。

## 背景信息（来自总计划，必须遵守）
- 颜色：主背景 #FFFFFF，二级背景 #F4F5F7，边框 #EBECF0，主文本 #172B4D，二级文本 #6B778C，强调色 #3B82F6，强调色 hover #2563EB。
- 字体：Inter，字号/字重按设计文档执行。
- 间距单位：8px。
- 卡片阴影与圆角必须按设计文档。

## 任务清单
- [ ] 在 `frontend/Modules/workspace/style.css` 定义全局 CSS 变量（颜色、字体、间距、阴影）。
- [ ] 实现卡片样式（历史列表/编辑区/评论浮窗），按钮主次样式，hover/active 过渡。
- [ ] 设定 workspace 布局为 25% / 75% 的 grid，gap 24px。
- [ ] 统一输入框/按钮的圆角与字体。
